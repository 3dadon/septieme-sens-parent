# Septième Sens

Septième Sens est un projet éditorial, sensoriel et numérique explorant ce qui donne du goût à la vie.

## Vision

Septième Sens explore les sensations, les matières, les images, les sons, les récits et l'intuition à travers une expérience digitale poétique, élégante et mobile-first.

## Structure du monorepo

Le monorepo conserve la structure existante du workspace :

- `septieme-sens-front/`
- `septieme-sens-back/`
- `septieme-sens-mobile/`
- `AGENTS.md`

## Applications

### septieme-sens-front

Frontend web de la V1, orienté expérience éditoriale, sensorielle et mobile-first.

### septieme-sens-back

Backend API prévu pour les besoins futurs du projet.

### septieme-sens-mobile

Base prévue pour une future application mobile.

## Documentation IA

Les fichiers `AGENTS.md` documentent les consignes de travail pour les agents IA au niveau du workspace et des sous-projets.

## Installation

Depuis la racine du monorepo :

```bash
cd septieme-sens-front
npm install
npm run dev
```
